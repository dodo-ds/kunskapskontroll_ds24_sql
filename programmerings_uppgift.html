1. Deskriptiv sammanfattning över databasen AdventureWorks2022¶

Databasen tillhör det fiktiva företaget AdventureWorks som är verksamt inom försäljning av cycklar, reservdelar och diverse tillbehör. Den är omfattande och strukturerad i fem olika scheman, där varje schema innehåller flera tabeller som tillsammans täcker alla aspekter av företagets verksamhet.¶

HumanResources¶

Detta schema innehåller detaljerad information om alla anstälda. Här kan vi hitta data om deras namn, lön, anställningsdatum, position och annan relevant information inom företaget.

Sales¶

Som namnet tyder på är detta schema dedikerat till data om allt relaterad till försäljning.

Person¶

Alla personer associerat med företaget så som anställda, försäljare, kunder och leverantörer.

Production¶

Allt som produceras och säljs. Det omfattar information om produkter, produktkategorier, komponenter osv.

Purchasing¶

Detta schema hanterar relationen med företagets leverantörer.

female_var = sales_by_emp_df[sales_by_emp_df["Gender"] == "F"]["TotalSalesYTD"].var()
female_n = sales_by_emp_df[sales_by_emp_df["Gender"] == "F"]["TotalSalesYTD"].count()

male_n = sales_by_emp_df[sales_by_emp_df["Gender"] == "M"]["TotalSalesYTD"].count()

# Beräkna s2p
s2p = ((female_n - 1) * female_var + (male_n - 1) * male_var) / (female_n + male_n - 2)

# Beräkna Standard Error
se = np.sqrt(s2p * (1 / female_n + 1 / male_n))

# Frihetsgrader
df = female_n + male_n - 2

# Signifikans nivå 95%
t_krit = t.ppf(1 - 0.025, df)

ci_lower_1 = (female_mean - male_mean) - t_krit * se
ci_upper_1 = (female_mean - male_mean) + t_krit * se

3. Executive summary¶

Översikt¶

  • Min analys fokuserar på prestandan och lönerna hos säljare på företaget, uppdelat mellan könen. Mitt mål var att undersöka om det fanns en statistiskt signifikant skillnad i prestation och ersättning mellan kvinnor och män inom företaget. Som källa användes företagets databas och statistiska tester genomfördes för att dra slutsatser.

Problem¶

  • Problemet som analyserades var att identifiera om företagets prestation och lönestruktur kan upplevas som rättvis. Denna analys är viktig för att säkerhetställa att företaget följer rättvisa riktlinjer och komma med rekommendationer för att undvika interna konflikter och negativt påverka de antställdas motivation.

  • Mina tester visar att det inte finns en statistiskt signifikant skillnad mellan kvinnor och män i varken prestation eller ersättning. Detta indikerar att AdventureWorks efterlever rättvisa riktlinjer när det gäller vår försäljningspersonal.

Rekommendationer¶

  • Trots att inga skillnader hittades, är det viktigt att företaget fortsätter på samma spår.

Nästa steg¶

  • Utföra regelbundna analyser för att säkerhetställa ingen regression införs över tid.
  • Analysera andra avdelningar om företagets riktlinjer efterlevs.
  • Anordna utbildningar inom jämställdhet och rättvisa för ledning och HR-personalen.
